# Algorithms Implemented

## 1. Minimax Algorithm

- Game tree search algorithm
- Used for Tic-Tac-Toe AI
- MAX and MIN player strategy
- Recursive search implementation

---

## 2. Alpha-Beta Pruning

- Optimized version of Minimax
- Eliminates unnecessary branches
- Improves search efficiency
- Tic-Tac-Toe implementation included

---

## 3. Heuristic Alpha-Beta

- Uses heuristic evaluation function
- Depth-limited search
- Faster than complete Minimax search
- Board evaluation using scores

---

## 4. Monte Carlo Tree Search (MCTS)

- Simulation-based search algorithm
- Uses random playouts
- Includes:
  - Selection
  - Expansion
  - Simulation
  - Backpropagation

---

## 5. Knowledge Graph

- Stores facts as triples
- Supports:
  - Adding facts
  - Querying relationships
  - Entity descriptions
  - Relationship searches

Example:

```text
(Paris) --[locatedIn]--> (France)
```

---

## 6. Bayesian Networks

- Probabilistic graphical model
- Implements:
  - Prior Sampling
  - Rejection Sampling
  - Exact Inference
- Weather prediction example included

---

## 7. AI Travel Planner

- Recommends destinations
- Builds travel itineraries
- Suggests food and drinks
- Calculates estimated trip cost
